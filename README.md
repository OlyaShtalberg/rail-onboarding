# RAiL Onboarding Bundle

Артефакты для нового разработчика команды RAiL (Rezolve AI Lab).

## Файлы

| Файл | Что внутри |
|---|---|
| `onboarding-rail.md` | Главный документ онбординга: доступы, каналы, созвоны, working norms, команда |
| `glossary.md` | Glossary имён и терминов Rezolve — для AI-агента (slim, без личных данных) |
| `general-skills.md` | _TBD_ — общие AI-скиллы для Cursor / Claude Code |
| `team-skills.md` | _TBD_ — внутренние скиллы команды RAiL |

## Как использовать

1. Прочитать `onboarding-rail.md` — стартовая точка
2. Подключить `glossary.md` как контекст-артефакт в Cursor / Claude (Project Knowledge / Skill / etc) — агент сможет отвечать на вопросы про команду, термины, продукты
3. Подключить `general-skills.md` и `team-skills.md` как доступные скиллы
